# mirr.OS one Display App

## Project setup

```
yarn install # installs dependencies to node_modules
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Lints and fixes files

```
yarn run lint
```

### Runs unit and snapshot tests

```
yarn run test:unit
```
